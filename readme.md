![psVRC](https://github.com/rainy-aa/psVRC/blob/main/psvrc.png)

unzip this into a folder where you want to put this

run install.ps1 if you want to be able to launch this from your taskbar, otherwise just run psVRC.ps1 directly

supports:
* setting custom icon on shortcut
* detects existing profiles to migrate from VRC Quick Launcher
* setting friendly name for each profile
* toggle for watch mode (watch world/avatar test builds) 
* toggle for vr/desktop
